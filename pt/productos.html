Sistema DUS-50

Sistema KUS-50

Sistema CBTU-50

O sistema MURZAN CBTU-50 foi projetado especificamente para produtos da indústria avícola, ideal para transferência de peitos de frango marinado. O sistema é composto de uma bomba de duplo diafragma, que prove um fluxo constante e de alta capacidade de produção.

Sistema PSW-70

Sistema D-150
